# 需求问诊室前端

这里是需求问诊室 / Requirements Clinic 的前端应用，负责展示首页、快速问诊、需求简报、正式项目地图和表达训练界面。

前端的主要目标是让用户在不理解需求分析方法的情况下，也能顺着界面完成澄清过程：从选择模式、回答问题、查看可视化整理，到生成概述、详细报告或训练反馈。

## 页面范围

- 首页：展示快速问诊、正式项目、表达训练三个同级入口。
- 快速问诊：从一句话或示例体验进入连续追问流程。
- 需求简报：展示普通概述和专业详细报告。
- 正式项目：用需求地图承载更复杂的项目分析。
- 表达训练：通过固定情境练习追问与总结。

## 本地预览

```bash
npm install
npm run dev
```

默认访问地址：`http://localhost:3000`。

未连接后端时，前端会使用本地示例数据，方便直接体验界面和交互；连接后端后，真实模型调用仍只通过后端完成，前端不会保存或展示模型 API key。
